# Barra de Progresso

## 📖 Introdução 

Este é um projeto de estudo prático em JavaScript, utilizando DOM para a criação de uma aplicação de uma Calculadora de Gorjeta. Projeto desenvolvido utilizando HTML, CSS e Javascript.
